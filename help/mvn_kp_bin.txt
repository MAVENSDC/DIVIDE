**************
* MVN_KP_BIN *
**************

  This routine will rebin a Key Parameter from the input array 
  in up to eight dimensions. 

mvn_kp_bin, kp_data, to_bin, bin_by, output, std_out, $
            binsize=binsize, list=list, avg_out=avg_out, mins=mins, maxs=maxs, $
            std = std, density=density, help=help

REQUIRED FIELDS
**************
  kp_data: In-situ Key Parameter Data Structure
  to_bin: The Key Parameter which will be binned
  bin_by: Up to eight key parameter indices or names 
          by which to bin the requested key parameter
  output: The requested Key Parameter binned in the desired dimensions. 
          By default, this is the the number of data points within each bin.
  std_out: Output array containing the standard deviations of the binned 
           Key Parameter data in each bin.

OPTIONAL FIELDS
***************
    mins: Optional minimum values for each of the binning dimensions
    maxs:  Optional maximum values for each of the binning dimensions
    binsize: Optional array defining the binsize to use for each of 
             the binning dimensions
    output: The requested Key Parameter binned in the desired dimensions. 
            By default, this is the the number of data points within each bin.
    std_out: Output array containing the standard deviation of the binned 
             key parameter in each bin.
    avg_out: Output array containing the average value of the 
             binned key parameter in each bin'
    density:  An output array containing the density of the binned parameter
    std: With this keyword, the routine will calculate the standard deviation 
         within each bin and return in in std_out
    list: Used to print out the contents of the input data structure.
    median: An output array containing the median value of each bin.
    help: Invoke this list.
