*************
* MVN_KP_3D *
*************

Interactive 3D visualization of MAVEN spacecraft trajectory and 
insitu/iuvs KP parameters.'

pro MVN_KP_3D, insitu, iuvs=iuvs, time=time, basemap=basemap, grid=grid, $
               cow=cow, tron=tron, subsolar=subsolar, submaven=submaven, $
               field=field, color_table=color_table, bgcolor=bgcolor, $
               plotname=plotname, color_bar=color_bar,axes=axes,$
               whiskers=whiskers,parameterplot=parameterplot,$
               periapse_limb_scan=periapse_limb_scan, direct=direct, $
               ambient=ambient,view_size=view_size, camera_view=camera_view, $
               mso=mso, sunmodel=sunmodel, optimize=optimize, $
               initialview=initialview, drawid=drawid, $
               scale_factor=scale_factor, spacecraft_scale=spacecraft_scale, $
               speckle=speckle, help=help
 
REQUIRED INPUTS
***************
  insitu: MAVEN In-situ KP data structure.

OPTIONAL KEYWORDS
*****************
  iuvs: MAVEn IUVS KP data structure
  time: TBD
  subsolar: if selected, will plot the subsolar location with a yellow disk.
  sunmodel: if selected, will plot the Mars-Sun vector as a yellow line.
  submaven: if selected, will plot the sub S/C location with a blue disk.
  grid: if selected, overplot a lat/lon grid with 30/45 deg res respectively
  mso: if selected, plot the S/C track and data in MSO coordinates.
  axes: if selected, plot the XYZ axes in the widget window.  MSO or GEO.
  optimize: For large data structures, the plotting of the orbital track can 
            get very slow. This keyword decimates the track to a managable size.
  direct: Generate visualization in direct graphics; but with no 
          interactive capabilities.
  basemap: The name of the basemap to display on which the data will be plotted
           If not included, a basic lat/lon grid is shown
           Choices include:
          'MDIM': The Mars Digital Image Model.
          'MOLA': Mars Topography in color.
          'MOLA_BW': Mars topography in black and white.
          'MAG':  Mars crustal magnetism.
          'DUST': IUVS Apopase Dust index image.
          'OZONE': IUVS Apopase Ozone index image.
          'RAD_H': IUVS Apopase H Radiance image.
          'RAD_O': IUVS Apopase O Radiance image.
          'RAD_CO': IUVS Apopase CO Radiance image.
          'RAD_NO': IUVS Apopase NO Radiance image.
          'USER': User definied basemap. Will open a file dialog window. 
  color_table: 1 or 4 element integer to provide color table info.
  bgcolor: 1 or 3 element byte to provide background color
  color_bar: 3-element byte to define text color in plotting subwindow
  ambient: Define the intensity of the flashlight that mimics sunlight.
  initialview: Define the initial view of the planet.  The fields are:
               latitude, longitude, radius [,x offset, y offset]
  scale_factor: Scale down the widget window for smaller screens.  
  spacecraft_scale: in, optional, Change the scale size of the MAVEn S/C.  
                                  Default value is 0.03

  help: Invoke this list.
